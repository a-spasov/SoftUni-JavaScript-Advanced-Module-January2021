01. Fruit
Write a function that calculates how much money you need to buy fruit. You will receive a string for the type of 
fruit you want to buy, a number for weight in grams and another number for the price per kilogram. 
Print the following text on the console:  'I need ${money} to buy {weight} kilograms {fruit}.'
Print the weight and the money rounded to two decimal places. The input comes as three arguments passed to your 
function. The output should be printed on the console.

02. Greatest Common Divisor - GCD
Write a function that takes two positive numbers as input and compute the greatest common divisor.	
The input comes as two positive integer numbers. The output should be printed on the console.
