01. Edit Element 
Create function edit() that takes three parameters.
First parameter is a reference to an HTML element, other two parameters are string – match and replacer.
You have to replace all occurrences of match inside the text content of the given elements with replacer.

02. Collect List Items
Write a JS function that scans a given HTML list and appends all collected list items’ text to a text area on the 
same page when the user clicks on a button.

03. Sum Numbers
Write a JS function that reads two numbers from input fields in a web page and puts their sum in another field 
when the user clicks on a button.

04. Show More
Write a JS function that expands a hidden section of text when a link is clicked. The link should disappear as the 
rest of the text shows up.
